# 图书图片管理系统项目方案
## 一、项目概述
本项目基于C# WinForm开发，实现图书信息录入、重复ISBN校验、本地图片批量压缩功能，采用SQLite轻量数据库存储图书数据，界面可视化操作，满足课程实验要求。

## 二、功能需求
1. 图书信息录入：支持输入ISBN、书名、作者、价格，提交存入数据库
2. 重复校验：相同ISBN不可重复新增，弹出拦截提示
3. 文件浏览：读取程序目录下图片文件展示在列表
4. 图片批量压缩：选中图片自动缩放尺寸，生成压缩副本
5. 弹窗提示：操作成功/失败均给出文字弹窗反馈

## 三、技术栈
- 开发语言：C# .NET Framework
- 界面框架：WinForm
- 数据库：SQLite
- 第三方依赖：System.Data.SQLite.Core
